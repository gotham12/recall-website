# Recall — Marketing Website

Premium landing page for [Recall](https://almightytamer.github.io/recall/) — AI-native cognitive care.

## Develop

```bash
npm install
npm run dev
```

## Deploy

Optimized for [Vercel](https://vercel.com). Push to GitHub and import the repo, or:

```bash
npx vercel --prod
```
